Quick Reference

Docker

  • docker -v show what version of docker you have installed
  • docker ps : List running containers. -a will list all containers
  • docker run <name of image> runs an instance of a container. -p <port number> to run on specific port
  • docker stop <name of container> stops a currently running container. it stops but keeps state. Can be restarted
  • docker rm <name of container> remove a nonrunning container from the list of containers
  • docker exec <name of container> <command> runs a command on a specific container
  • docker swarm init initialize a machine as a swarm manager
  • docker tag <image> associate a docker image with a repository
  • docker push <image> push an image to a repository

Docker Compose

  • docker-compose up run the multi-service app in the current directory
  • docker-compose can run most basic docker commands including the ones above

Docker Machine

  • docker-machine create --driver virtualbox <name of VM> creates a virtual machine running docker
  • docker-machine ssh <name of VM> <docker command> runs a docker command on a virtual machine
  • docker-machine ls lists the currently running VMs
  • docker-machine env <VM name> gives the command to switch docker environment to execute commands on that VM
  • docker-machine start <VM name> restart a stopped VM
